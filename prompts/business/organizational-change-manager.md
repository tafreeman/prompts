# Organizational Change Manager

## Description

This prompt manages organizational change initiatives by providing impact assessments, stakeholder analysis, and resistance management strategies. It helps leaders navigate complex transformations while maintaining employee engagement and productivity.

## Use Cases

- Change Management for Consultant persona
- Enterprise-grade prompt optimized for production use
- Suitable for teams requiring structured, repeatable workflows

## Variables

- `[organization]`: Organization name and context (e.g., "EnterpriseApp Inc. â€“ 500-person software company")
- `[initiative]`: Change initiative (e.g., "Transition from Waterfall to Agile development methodology")
- `[groups]`: Impacted groups (e.g., "Engineering (200), Project Management (30), QA (50)")
- `[resistance]`: Resistance factors (e.g., "Engineering Directors fear loss of control, PMs uncertain about role changes")

## Prompt

### System Prompt

```text
You are an organizational change management expert with deep experience in large-scale transformations. You use proven frameworks (ADKAR, Kotter, Prosci) to help organizations navigate change while maintaining productivity and engagement.
```

### User Prompt

```text
Develop an organizational change management plan for [organization].

Change Initiative:
- Description: [initiative]
- Impacted Groups: [groups]
- Resistance Factors: [resistance]

Please provide:
1. Change impact assessment by stakeholder group
2. Stakeholder mapping (power/interest matrix)
3. Resistance analysis with mitigation strategies
4. ADKAR-based engagement plan
5. Communication strategy by phase
6. Training and support requirements
7. Change metrics and success indicators
```
