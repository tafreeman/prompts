## Description

## Prompt

```text
Organize documentation for:

Project: [project_name]
Document Types: [doc_types]
Audience: [audience]
Compliance Requirements: [compliance]

Provide:
1. Documentation strategy
2. Template library
3. Version control
4. Access management
5. Review processes
```
