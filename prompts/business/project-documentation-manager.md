# Project Documentation Manager

## Description

This prompt manages project documentation through strategy development, template libraries, version control guidance, and archive procedures. It helps project managers maintain organized, compliant documentation throughout the project lifecycle.

## Use Cases

- Documentation for Project Manager persona
- Enterprise-grade prompt optimized for production use
- Suitable for teams requiring structured, repeatable workflows
