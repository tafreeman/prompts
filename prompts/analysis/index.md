# Analysis & Research Prompts

Turn raw interviews, spreadsheets, and telemetry into decision-ready insight packs. These prompts help PMs, researchers, and analysts compress discovery cycles while keeping audit trails of sources and assumptions.

## In This Section

| Objective | Prompt | Deliverable |
| :--- |--------| :--- |
| Build a go/no-go case | [Business Case Developer](/prompts/analysis/business-case-developer) | ROI summary, risks, KPIs |
| Map competitors | [Competitive Intelligence Researcher](/prompts/analysis/competitive-intelligence-researcher) | Feature matrix + SWOT |
| Design KPIs | [Metrics & KPI Designer](/prompts/analysis/metrics-and-kpi-designer) | SMART metric proposal |
| Improve workflows | [Process Optimization Consultant](/prompts/analysis/process-optimization-consultant) | Current vs. future state |
| UX research | [User Experience Analyst](/prompts/analysis/user-experience-analyst) | Persona-level findings |

## Quick Picks

- **Need interview synthesis?** Use [Stakeholder Requirements Gatherer](/prompts/analysis/stakeholder-requirements-gatherer) for traceable requirement logs.
- **Quality or data drift?** [Data Quality Assessment](/prompts/analysis/data-quality-assessment) highlights anomalies and remediation steps.
- **Libraries at a glance?** The visualization prompts ([Library Capability Radar](/prompts/analysis/library-capability-radar), [Network Graph](/prompts/analysis/library-network-graph)) turn prompt metadata into charts.
