## Using Analysis Prompts

1. **Choose a prompt** for your analysis type
2. **Provide data context** (type, size, time period)
3. **Include key metrics** you're tracking
4. **State your goals** clearly
5. **Share relevant findings** or hypotheses

## Tips for Analysts

- Describe your data structure and size
- Include summary statistics when possible
- Mention the business context
- Clarify what decision the analysis will inform
- Specify your audience (technical vs. executive)
- Include any known issues or anomalies
- Request specific visualization types if preferred

## Data Privacy Note

⚠️ **Important**: Never share personally identifiable information (PII), confidential data, or sensitive business information in prompts. Use:

- Aggregated data
- Sample data
- Anonymized examples
- Summary statistics
- Mock data for illustration

## Getting Better Insights

**Provide this context:**

- Industry or business type
- Time period covered
- Sample size
- Key metrics tracked
- Current challenges
- Decision to be made

**Ask for:**

- Executive summaries
- Specific recommendations
- Statistical significance
- Visualization suggestions
- Follow-up analysis ideas

## Statistical Significance

When analyzing data:

- Mention sample sizes
- Ask about statistical significance
- Request confidence intervals
- Question correlations vs. causation
- Consider external factors
