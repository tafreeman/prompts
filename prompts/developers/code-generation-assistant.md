# Code Generation Assistant

## Description
Generates clean, efficient code based on requirements

## Use Cases
- Code Generation for Developer persona
- Enterprise-grade prompt optimized for production use
- Suitable for teams requiring structured, repeatable workflows

## Prompt

```
Generate [language] code for the following requirements:

Functionality: [functionality]
Input: [input_format]
Output: [output_format]
Constraints: [constraints]

Please provide:
1. Clean, well-commented code
2. Error handling
3. Unit tests
4. Documentation
5. Performance considerations
```

## Variables
- `[constraints]`: Constraints
- `[functionality]`: Functionality
- `[input_format]`: Input Format
- `[language]`: Language
- `[output_format]`: Output Format

## Example Usage

**Input:**
Replace the bracketed placeholders with your specific values, then use with Claude Sonnet 4.5 or Code 5.

**Output:**
The AI will provide a comprehensive response following the structured format defined in the prompt.

## Tips
- Be specific when filling in placeholder values for better results
- Review and adjust the output to match your organization's standards
- Use this as a starting template and refine based on feedback
- For best results, provide relevant context and constraints
