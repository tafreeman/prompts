# Developer Prompt Library

Ship faster with curated prompts for coding, reviews, refactoring, DevOps, and documentation. These templates are optimized for GitHub Copilot Chat, Claude, and GPT so you can describe intent in natural language and let the AI scaffold the rest.

## In This Section

| Workflow | Prompt(s) | What You Get |
|----------|-----------|--------------|
| **New features** | [Code Generation Assistant](/prompts/developers/code-generation-assistant) | Ready-to-run scaffolds with inline TODOs |
| **API design** | [API Design Consultant](/prompts/developers/api-design-consultant), [Dotnet API Designer](/prompts/developers/dotnet-api-designer) | Endpoint specs, DTOs, and sample requests |
| **Code review** | [Code Review Assistant](/prompts/developers/code-review-assistant), [Structured Code Review](/prompts/developers/code-review-expert-structured) | Repeatable review formats with severity tagging |
| **Refactoring** | [Refactoring Plan Designer](/prompts/developers/refactoring-plan-designer), [C# Refactoring Assistant](/prompts/developers/csharp-refactoring-assistant) | Stepwise plan + sample commits |
| **Ops & Quality** | [DevOps Pipeline Architect](/prompts/developers/devops-pipeline-architect), [Test Automation Engineer](/prompts/developers/test-automation-engineer) | CI/CD pipelines, coverage plans |
