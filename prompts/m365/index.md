# M365 Copilot Prompt Pack

Accelerate exec updates, knowledge handoffs, and daily communications across the Microsoft 365 suite. Every prompt includes role guidance and paste-ins for Word, Outlook, PowerPoint, Excel, and Designer so you can copy, tweak variables, and ship in minutes.

## In This Section

| App | Prompt | Output |
|-----|--------|--------|
| Outlook | [Email Triage Helper](/prompts/m365/m365-email-triage-helper) | Prioritized inbox with draft responses |
| Teams/Loop | [Daily Standup Assistant](/prompts/m365/m365-daily-standup-assistant) | Digest with blockers + action requests |
| Word | [Handover Document Creator](/prompts/m365/m365-handover-document-creator) | SOP + ownership table |
| PowerPoint | [Presentation Outline Generator](/prompts/m365/m365-presentation-outline-generator) | Slide storyline w/ talking points |
| Excel | [Excel Formula Expert](/prompts/m365/m365-excel-formula-expert) | Function guidance and examples |
| Designer/Sway | [Designer Social Media Kit](/prompts/m365/m365-designer-social-media-kit) | Image briefs + copy |

## Workflow Bundles

- **Meetings:** Prep with [Meeting Brief](/prompts/m365/m365-meeting-prep-brief), capture actions via [Meeting Recap Assistant](/prompts/m365/m365-meeting-recap-assistant), and file notes using [Weekly Review Coach](/prompts/m365/m365-weekly-review-coach).
- **Status Reporting:** Combine [Project Status Reporter](/prompts/m365/m365-project-status-reporter) and [Slide Content Refiner](/prompts/m365/m365-slide-content-refiner) to go from raw notes to a polished deck.
- **Design & Comms:** Use the Designer prompts for visuals, then adapt copy with [Sway Visual Newsletter](/prompts/m365/m365-sway-visual-newsletter).

## Tips

- Replace bracketed placeholders (e.g., `[TEAM NAME]`) before pasting into Copilot for context-rich responses.
- Pair with Viva Insights or Planner exports for more accurate data references.
