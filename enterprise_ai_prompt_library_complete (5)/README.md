## Technology Stack

### Backend
- **Framework**: Flask (Python)
- **Database**: SQLite
- **Server**: Development server (Flask built-in)

### Frontend
- **CSS Framework**: Custom design system (Stage 3) / Bootstrap 5 (Stages 1-2)
- **Typography**: Inter font family (Stage 3) / System fonts (Stages 1-2)
- **Icons**: Font Awesome 6
- **JavaScript**: Vanilla JS with modern ES6+

## Installation & Setup

### Prerequisites
- Python 3.7+
- pip (Python package manager)

### Quick Start (Any Stage)
```bash
# Navigate to desired stage
cd enterprise_ai_prompt_library_development/stage_X_name/

# Install dependencies
pip install flask

# Load data (Stages 2 & 3 only)
python load_expanded_data.py

# Run application
python app.py

# Access at http://localhost:5000
```
