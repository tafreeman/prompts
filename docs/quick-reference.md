Every prompt includes:

- **Variables**: What to replace

## üî• Pro Tips

### Iterate, Don't Perfect
First response = starting point. Ask for changes:
- "Make it shorter"
- "Add more examples"
- "Use simpler language"

### Save What Works
Keep a personal collection of customized prompts that work for your needs.

### Combine Prompts
Use multiple prompts for complex projects:
1. Strategy prompt ‚Üí Direction
2. Creative prompt ‚Üí Content
3. Analysis prompt ‚Üí Validation

### Provide Examples
Show what you want: "Write it like this: [example]"

### Be Conversational
Talk to AI like a colleague:
- "That's good, but can you make it more technical?"
- "I need this for executives, simplify it"

## üõ†Ô∏è When Prompts Don't Work

**Problem**: Response is too generic
**Fix**: Add more specific details and context

**Problem**: Wrong tone or style
**Fix**: Specify tone explicitly ("professional", "casual", "technical")

**Problem**: Too long or too short
**Fix**: State exact length requirements

**Problem**: Missing key points
**Fix**: List specific points to cover

- Filter by difficulty
- Check version history
