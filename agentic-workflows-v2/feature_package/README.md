# Feature Package

This directory stores generated or handoff-ready feature artifacts.

## Files

- `feature_package/manifest.json`: machine-readable package metadata

## Intended Use

Use this folder as a staging area for outputs from fullstack/codegen workflows when preparing implementation handoff to developers or reviewers. Each handoff should commit `manifest.json` plus any artifacts specific to that handoff; freeform handoff notes belong in the PR description, not in a persistent file.
